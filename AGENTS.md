# Proyecto: App de Gestión de Comunidades (MVP)

## Tecnologías
- Backend: Node.js + Express + PostgreSQL (o SQLite para desarrollo)
- Frontend: React + Vite (o la que prefieras)
- Autenticación: JWT
- API: RESTful

## Módulos del MVP (priorizados)
1. Autenticación (registro/login/roles)
2. Dashboard (vista general para residente y admin)
3. Gestión de expensas (crear, listar, pagar, confirmar)
4. Comunicación (anuncios y tickets)
